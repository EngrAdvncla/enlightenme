~Ito po yung course na ang main focus ay hardware at software. Ito yung course na magbibigay hatid kung paano pagkasunduin ang hardware at software. Hindi po kasi pwedeng software to hardware and vice-verse nang ganun-ganun lang. Kailangan muna nilang magkaintindihan, at para magkaintindihan sila kailangan nila ng translator. Dito kailangan din ng math at logic, kasi kayo ang gagawa ng paraan para magkaintindihan ang software, hardware, at peopleware. Dito kayo ang magdedesign ng pinakamabisang logic para gumana ng maayos at mabilis ang mga application. Dito pwede kayong maging OS developer or sa robotics/automation. Dito ang kailangan ang wit and creativity. Ang ibig sabihin para maging maayos ang pamumuhay mo sa engineering, kailangan mong mag explore, mag try at higit sa lahat be hands-on. Dito po ay create, create, then create.
